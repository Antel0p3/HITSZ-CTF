    web相关的技术资料有很多，而且我们能够google到非常多的姿势。如何将现有的知识以一条完整的链组合起来，是我认为的web的核心，需要坚持不懈的努力形成比较完整的渗透体系。同时，web它又很多很杂，上手应该是最简单的，继续学下去如果不做总结的话肯定会被淹没。以下是本人半年以来的知识点总结，希望能够帮助大家的学习，欢迎各位web选手的补充。鉴于本人能力水平有限，如发现错误请先告知，然后修改。最后，总结可能会有部分重复内容放在不同的模块里面，这是因为我个人希望尽可能的将解题重心放在思路方面。

    然后是本总结是一个框架性质的东西，不可能作为百科一样面面俱到，但能够方便大家对知识做好管理。（如果有时间的话，我也会写一点好玩的东西）

    再有就是一些工具性的东西我会选择性的介绍，但是最基础的burpsuite，SwitchyOmega等这些东西还是默认要会的。（不会的自己写点总结放上来）
