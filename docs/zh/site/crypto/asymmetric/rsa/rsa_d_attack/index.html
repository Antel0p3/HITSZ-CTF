    私钥 d 相关攻击¶

    d 泄露攻击¶

    攻击原理¶

    首先当 d 泄露之后，我们自然可以解密所有加密的消息。我们甚至还可以对模数 N 进行分解。其基本原理如下

    我们知道 ed \equiv 1 \bmod \varphi(n)，那么存在一个 k 使得

    ed-1=k\varphi(n)

    又 \forall a\in {Z}_n^*，满足a^{ed-1}\equiv1(\bmod n)。令

    ed-1=2^st

    其中，t 是一个奇数。然后可以证明对于至少一半的 a\in {Z}_n^*，存在一个 i\in[1,s]，使得

    a^{2^{i-1}t}\not\equiv\pm1(\bmod n),a^{2^{i}t}\equiv1(\bmod n)

    成立。如果 a,i 满足上述条件，gcd(a^{2^{i-1}t}-1,n)是 n 的一个非平凡因子，所以可以对 n 进行暴力分解。

    工具¶

    利用以下工具可以直接进行计算

    • RsaConverter.exe (https://sourceforge.net/projects/rsaconverter/ , for windows )
    • rsatool.py(分解原理如上)

    Wiener-Attack 及其扩展¶

    条件：已知n, e，且e较大或d 比较小（d<\frac{1}{3}N^{\frac{1}{4}}）

    例如以下little_trick阶段d1是256位，n是2048位，说明e较大，可以直接跑脚本得到d

    from Crypto.Util.number import *
from libnum import *
from secret import flag,special,p,q,n


def little_trick(msg):
    p1 = getPrime(1024)
    q1 = getPrime(1024)
    n1 = p1 * q1
    d1=random.randint(1,2**256)
    e1=inverse(d1,(p1-1)*(q1-1))
    print(n1)
    print(e1)
    print(pow(msg,e1,n1))


def real_trick():
    assert (special > (ord("*")*100) and gcd(special,(p-1)*(q-1))!=1 )
    print(n)
    print(pow(libnum.s2n(flag),special,n))


if __name__ == '__main__':
    little_trick(p-1)
    real_trick()

    攻击原理¶

    连分数¶

    渐进分数¶

    渐进分数是指一个以分数的形式出现的两个有理数的商的近似值，如我们熟知的密率355/113和约率22/7就是3.1415926/1的渐进分数.

    连分数展开的渐进分数概念有些不同，例如\dfrac{543}{236}的连分数展开为

    2+\frac{1}{3+\frac{1}{3+\frac{1}{11+1/2}}}

    则第一个渐进分数是 2，第二个渐进分数是 2+\dfrac 13，第三个渐进分数是 2+\dfrac{1}{3+1/3}，以此类推.

    定理¶

    Wiener Attack基于以下定理：

    若 \xi 是一个实数，a,b 互素，且满足

    |\xi -\frac ab| \lt \frac 1{2b^2}

    则 \dfrac ab 是 \xi 的连分数展开的一个渐进分数.

    Wiener Attack 的条件是 q<p<2q,~d<\frac 13N^{1/4}，由 ed-k\phi(N)=1，有

    \lvert \frac{e}{\phi(N)} - \frac kd\rvert=\frac 1{d\phi(N)}

    因为\dfrac 1{d\phi(N)} 很小，所以可以认为\dfrac kd是\dfrac e{\phi(N)}的近似，且在p,q很大时，\phi(N)和N接近.

    因为\phi(N)=N-p-q-1 且 p+q-1<3\sqrt N（应该是由 q<p<2q 推出的），我们得到 |N-\phi(N)|<3\sqrt N.

    用 N 代替 \phi(N)，可以得到：

    \begin{align} |\frac eN - \frac kd|&=|\frac {ed-k\phi(N)-kN+k\phi(N)}{Nd}|\\ &=|\frac {1-k(N-\phi(N))}{Nd}|\\ &\le |\frac {3k\sqrt N}{Nd}|=\frac {3k}{d\sqrt N} \end{align}

    因为 k\phi(N)=ed-1 < ed，且 e <\phi(N)，所以 k<d<\frac 13 N^{1/4}. 则

    |\frac eN - \frac kd| \le \frac 1{d\sqrt[4]{N}}<\frac 1{2d^2}

    由上面的定理可以得到，\dfrac kd 是 \dfrac eN 的连分数展开的一个渐进分数，即 \dfrac kd 就在连分数展开的渐进分数中.

    连分数展开可以多项式时间内完成，只需要再遍历各个渐进分数，设某一渐进分数为 \dfrac uv，判断 ev-1 是否整除 u（即 ed-1 是否整除 k），若能整除，则当前的 u,v 就是 k,d.

    扩展用法 [羊城杯 2020]RRRRRRRSA¶

    题目如下（为节省空间，删除了输出数据部分的代码，输出的是两组N, E, c

    import hashlib
import sympy
from Crypto.Util.number import *

flag = 'GWHT{************}'

flag1 = flag[:19].encode()
flag2 = flag[19:].encode()

    分析¶

    已知 N_1=p_1^2q_1, N_2=p_2^2q_2，则

    \frac {N_1}{N_2}=(\frac {p_1}{p_2})^2\frac {q_1}{q_2}<\frac {q_1}{q_ 2}<1

    已知 p_2-p_1<1000，可以认为 \dfrac {N_1}{N_2} 和 \dfrac {q_1}{q_2} 很接近，猜测

    |\frac {N_1}{N_2}-\frac {q_1}{q_2}|<\frac 1{2q_2^2}

    （应该可以证明，但是直接试试能不能解更省时间）

    于是对 \dfrac {N_1}{N_2} 进行连分数展开，发现渐进分数中有 \dfrac {q_1}{q_2}，之后即可解密.

# 将分数x/y转为连分数的形式
def get_contined_fraction(x, y):
    continued_fraction = []
    while y:
        continued_fraction.append(x // y)
        x, y = y, x % y
    return continued_fraction

# 输入：连分数列表
# 输出：连分数的渐进分数分子num和分母den
def get_convergent(continued_fraction):
    num, den = 1, 0
    # 从后面往前循环
    for i in continued_fraction[::-1]:
        den, num = num, i * num + den
    return den, num

# 求解每个渐进分数
def get_all_convergents(x, y):
    res = get_contined_fraction(x, y)
    # 将连分数的结果逐一截取以求渐进分数
    res = list(map(get_convergent, (res[0:i] for i in range(1, len(res)))))
    return res

def expand_wiener_attack(n1, n2):
    for (q2, q1) in get_all_convergents(n1, n2):  
        # 找到一个满足条件的渐进分数
        if q1 == 0:
            # 可能会出现连分数的第一个为0的情况，排除
            continue
        if n1 % q1 == 0 and q1 != 1:
            return (q1, q2)
    print("Failed!")

q1, q2 = expand_wiener_attack(N1, N2)
# print(f"q1 = {q1}\nq2 = {q2}")

assert N2 % q2 == 0
p1 = int(iroot(N1 // q1, 2)[0])
p2 = next_prime(p1)
assert N2 % p2 == 0

phi1 = (p1 * p1 - p1) * (q1 - 1)
phi2 = (p2 * p2 - p2) * (q2 - 1)
d1 = invert(E1, phi1)
d2 = invert(E2, phi2)

m1 = n2s(int(powmod(c1, d1, N1)))
m2 = n2s(int(powmod(c2, d2, N2)))
m = m1 + m2
print(m)

    def get_ed(p, q):
    k = cal_bit(q*p)
    phi_n = (p-1)*(q-1)
    r = random.randint(10, 99)
    while True:
        u = getPrime(k/4 - r)
        if gcd(u, phi_n) != 1:
            continue
        t = invmod(u, phi_n)
        e = pi_b(t)

    d = invmod(e, phi_n)
    return (e, d)

    可以看出，我们得到的 u 的位数比 n 的位数的四分之一还要少，这里其实就差不多满足了 Wiener's Attack 了。而且我们计算出来的 u，t，e，d 还满足以下条件

    \begin{align*} ut &\equiv 1 \bmod \varphi(n) \\ et &\equiv 1 \bmod bt \\ ed &\equiv 1 \bmod \varphi(n) \end{align*}

    根据题中给出的条件，我们已经知道了 n，e，bt。
