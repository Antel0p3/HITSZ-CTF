    看着WP复现（不是）

    同时稍微作点总结

    admin.php(buu上扫不了)

    admin + 12345弱口令（完全可以加个sql注入）

    然后就见到了我们的cms

    最开始准备试cms一些cve，在网上找到的CVE大多数都有些老，最新的也在6.x版本，而这个是7.x版本，不大用的了。

    然后手动爬站

    怀疑过这个

    怀疑这个是因为之前有过题目可以上传一个被篡改的主题然后显示拿到权限。

    然后并不是，但是有更为可疑的地方。

    可以编辑全局文件的头，可以看出{}包裹的nei容回被当成是php代码执行，类似某种模板

    点保存的时候给出了更为奇怪的路径，也就是说要创建一个这样的路径。

    由于目前路径在 /var/www/html/system/tmp/template/default/common/header.html.php之下，需要找个路径穿越的点。

    然后就有微信小程序那块

    最后在模板中添加{!echo(system('cat /flag'))}
